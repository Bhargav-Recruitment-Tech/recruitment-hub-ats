# Recruitment Hub ATS

Recruitment Hub ATS is a web-based application designed to manage recruitment workflows efficiently.

## Features
- User Login Authentication
- Candidate Management (Add, View, Edit, Delete)
- Live Search (Search candidates by Name, Email, Phone, Skills...) and Pagination
- Clean Dashboard UI
- SQLite Database Integration

## Tech Stack
- Python (Flask)
- SQLite
- HTML, CSS, Bootstrap

## Purpose
This project was built to demonstrate a real-world recruitment workflow system and hands-on development skills.

## Status
Active – continuously improving with new modules.
